# Exceptions Module E2E Scenarios

## 1. View All Exceptions

- **Precondition:** User is logged in, exceptions exist
- **Steps:**
  1. Navigate to Exceptions
- **Expected:** Exceptions list is visible

## 2. Filter/Search Exceptions

- **Precondition:** Multiple exceptions exist
- **Steps:**
  1. Enter search/filter criteria
- **Expected:** Only matching exceptions are shown

## 3. View Exception Details

- **Precondition:** Exception exists
- **Steps:**
  1. Click on exception
- **Expected:** Exception details are displayed
